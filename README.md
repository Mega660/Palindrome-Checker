# Palindrome Checker (Java)

## Detailed Description

The **Palindrome Checker** is a Java console application that determines whether a given input is a palindrome.

A palindrome is a word, number, or phrase that reads the same forward and backward.
Examples include "madam", "racecar", and "A man a plan a canal Panama"

The program improves accuracy by cleaning the input first:

* Removing spaces
* Removing punctuation
* Converting text to lowercase

This ensures phrases and sentences are checked correctly.

The project demonstrates several important Java programming concepts such as **string manipulation, regular expressions, and input handling

## Features

*  Accepts words, numbers, and phrases
*  Automatically removes spaces and punctuation
*  Case-insensitive palindrome checking
*  Fast string reversal using `StringBuilder`
*  Clear result output

## Technologies Used

* Java
* String Manipulation
* Regular Expressions (`replaceAll`)
* StringBuilder
* Scanner (User Input)

## How to Run

 Install Java (JDK 8 or higher)

 Compile the program:

javac PalindromeChecker.java

 Run the program:

java PalindromeChecker
